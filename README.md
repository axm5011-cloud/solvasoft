# موقع سولفا سوفت | Solvasoft

موقع تعريفي بصفحة واحدة (Single Page) مبني بـ **HTML + CSS + JavaScript** فقط — بدون أي إطار عمل أو خطوة بناء، جاهز مباشرة للرفع على **GitHub Pages**.

## الملفات
```
index.html              # الصفحة الكاملة
assets/css/style.css     # كل التنسيقات والأنميشن
assets/js/main.js        # التفاعلات (قائمة الجوال، ظهور العناصر، زر الأعلى…)
assets/img/logo.png      # الشعار
assets/img/icon.png      # الأيقونة (favicon)
.nojekyll                # يمنع GitHub من معالجة الملفات
```

## الرفع على GitHub Pages
1. أنشئ مستودعاً جديداً على GitHub وارفع كل الملفات في هذا المجلد إلى الجذر (root).
2. من المستودع: **Settings → Pages**.
3. في **Source** اختر الفرع `main` والمجلد `/ (root)` ثم **Save**.
4. بعد دقيقة سيكون الموقع متاحاً على:
   `https://<username>.github.io/<repo-name>/`

> إن استخدمت اسم مستودع `<username>.github.io` سيُنشر على الجذر مباشرة بدون مسار فرعي.

## معاينة محلية (اختياري)
```bash
python -m http.server 8090
# ثم افتح http://localhost:8090
```

## المميزات
- تصميم RTL عربي بثيم داكن مطابق للتصميم الأصلي.
- **تبديل لغة فعلي (عربي ⇄ إنجليزي)** بزر EN — يترجم كل النصوص ويعكس اتجاه الصفحة، ويحفظ اختيارك.
- **مؤشر ماوس مخصص** (نقطة + حلقة تتبعها) على أجهزة الكمبيوتر فقط، يتفاعل مع العناصر — ويختفي تلقائياً على الجوال/اللمس.
- أنميشن: ظهور العناصر عند التمرير، خلفية متحركة (Aurora)، تأثيرات hover على البطاقات، شريط علوي متغيّر، زر العودة للأعلى.
- متجاوب بالكامل (جوال / تابلت / كمبيوتر) مع قائمة جوال — بدون أي تجاوز أفقي.
- يحترم `prefers-reduced-motion` لمن يفضّل تقليل الحركة.
